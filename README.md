# 🌐 Creator Portfolio

> A professional portfolio website showcasing creative work and projects for content creators and designers.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📋 Overview

Creator Portfolio is a modern, visually appealing portfolio website designed specifically for creative professionals - designers, artists, videographers, and digital creators.

## ✨ Features

- 🎨 **Portfolio Gallery** - Showcase creative work with stunning layouts
- 📱 **Responsive Design** - Perfect on all devices
- 🎬 **Media Support** - Images, videos, and multimedia content
- 💼 **Professional Layout** - Modern, clean aesthetic
- 📊 **Statistics & Analytics** - Project metrics and achievements
- 🔗 **Social Integration** - Connect social media profiles
- ⚡ **Fast Performance** - Optimized loading times
- 🌙 **Dark Mode** - Modern color schemes

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with animations
- **JavaScript** - Interactive features
- **Font Awesome** - Professional icons
- **Google Fonts** - Beautiful typography

## 🚀 Quick Start

### Prerequisites
- Modern web browser
- Text editor
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/csgoowda/creator-portfolio.git
cd creator-portfolio

# Open index.html in browser
# Or use a local server

# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

## 📖 Usage

### Navigation
- **Home** - Hero section with introduction
- **Portfolio** - Gallery of creative work
- **About** - Creator background and bio
- **Contact** - Get in touch form
- **Social** - Social media links

### Adding Portfolio Items

Add new work to gallery:

```html
<div class="portfolio-item">
  <img src="assets/images/project1.jpg" alt="Project Description">
  <div class="portfolio-info">
    <h3>Project Title</h3>
    <p>Project description and details</p>
    <a href="#">View Project</a>
  </div>
</div>
```

### Customization

#### Update Creator Info

```html
<h1>Your Name</h1>
<p>Your Professional Title</p>
<p>Your bio and creative focus...</p>
```

#### Add Social Media

```html
<a href="https://instagram.com/yourprofile">Instagram</a>
<a href="https://behance.com/yourprofile">Behance</a>
<a href="https://dribbble.com/yourprofile">Dribbble</a>
```

#### Customize Colors

```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --accent-color: #your-color;
}
```
