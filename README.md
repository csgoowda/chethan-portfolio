# Personal Portfolio 🌐
Welcome to the source code of my digital home. This project showcases my journey as a Full Stack Developer, my freelance work, and my transition from Mechanical Engineering to Information Science.

[Live Demo Link Here]

### 🎨 Key Features
- **Project Gallery:** A curated display of my technical projects (EduSphere, Arunodhaya Cafe, etc.).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Modern UI/UX:** Built with a focus on clean aesthetics and smooth navigation.
- **Contact Integration:** Direct links for clients and recruiters to reach out.

### 🛠️ Tech Stack
- **Framework:** Next.js / React.js
- **Styling:** Tailwind CSS (for modern, utility-first design)
- **Animation:** Framer Motion / AOS (if applicable)
- **Deployment:** Vercel / Netlify

### 🏗️ Architecture
The portfolio is designed with a component-based architecture, ensuring that adding new projects or updating my resume is as simple as updating a JSON file.

### 🚀 Getting Started
To run this project locally:
1. Clone the repo: `git clone https://github.com/[your-username]/chethan-portfolio.git`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
